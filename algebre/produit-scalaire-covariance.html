Produit scalaire, Orthogonalité & Matrice de Covariance

Le chaînon manquant entre l'algèbre linéaire et l'ACP — trois notions fusionnées en une page.

Cette page : 1. Produit scalaire 2. Orthogonalité 3. Matrice de covariance 4. Centrage & réduction

1. Produit scalaire

Définition

Le produit scalaire de deux vecteurs \(\mathbf{u}\) et \(\mathbf{v}\) de \(\mathbb{R}^n\) est un scalaire qui mesure à quel point ils "pointent dans la même direction". C'est la base de toutes les notions de distance, angle et projection.

\[ \langle \mathbf{u}, \mathbf{v} \rangle = \mathbf{u}^T \mathbf{v} = \sum_{i=1}^{n} u_i v_i \]
u, v : vecteurs colonnes de \(\mathbb{R}^n\)  ·  Écriture matricielle : \(\mathbf{u}^T\mathbf{v}\) (vecteur ligne × vecteur colonne)
Lien avec l'angle : \(\langle \mathbf{u}, \mathbf{v} \rangle = \|\mathbf{u}\| \cdot \|\mathbf{v}\| \cdot \cos\theta\)

Norme d'un vecteur

La longueur de \(\mathbf{v}\) : \(\|\mathbf{v}\| = \sqrt{\langle \mathbf{v},\mathbf{v} \rangle} = \sqrt{\sum v_i^2}\)

Vecteur unitaire : \(\hat{\mathbf{v}} = \frac{\mathbf{v}}{\|\mathbf{v}\|}\) avec \(\|\hat{\mathbf{v}}\| = 1\)

Projection

Projection de \(\mathbf{u}\) sur \(\mathbf{v}\) :

\(\text{proj}_{\mathbf{v}}\mathbf{u} = \frac{\langle \mathbf{u},\mathbf{v}\rangle}{\|\mathbf{v}\|^2}\mathbf{v}\)

Utilisé pour calculer les composantes principales en ACP.

2. Orthogonalité & base orthonormée

Définition

Deux vecteurs sont orthogonaux si leur produit scalaire est nul : \(\langle \mathbf{u}, \mathbf{v} \rangle = 0\), c'est-à-dire \(\theta = 90°\). Une base est orthonormée si tous ses vecteurs sont orthogonaux deux à deux et de norme 1.

\[ \langle \mathbf{e}_i, \mathbf{e}_j \rangle = \delta_{ij} = \begin{cases} 1 & \text{si } i = j \\ 0 & \text{si } i \neq j \end{cases} \]
Condition d'une base orthonormée. En écriture matricielle : si \(P\) a des colonnes orthonormées, alors \(P^T P = I\), donc \(P^{-1} = P^T\). C'est exactement la propriété des vecteurs propres de la matrice de covariance en ACP.

Matrice orthogonale

\(P\) est orthogonale si \(P^T P = I\), soit \(P^{-1} = P^T\).

Propriété : une transformation orthogonale conserve les distances et les angles.

Pourquoi c'est capital en ACP

Les vecteurs propres d'une matrice symétrique (comme la matrice de covariance) sont toujours orthogonaux. On peut donc toujours construire une base orthonormée → \(P^{-1} = P^T\).

3. Matrice de variance-covariance

Pourquoi ?

La matrice de covariance résume toutes les relations linéaires entre les variables d'un dataset. C'est l'objet central de l'ACP : on la diagonalise pour trouver les axes de variance maximale. Sa construction matricielle est souvent demandée à l'examen.

\[ \Sigma = \frac{1}{n} X_c^T X_c \]
X_c : matrice des données centrées (\(n\) individus × \(p\) variables)  ·  n : nombre d'individus  ·  L'élément \(\sigma_{ij}\) = covariance entre la variable \(i\) et la variable \(j\).
La diagonale contient les variances : \(\sigma_{ii} = \text{Var}(X_i)\).
Hors diagonale : covariances : \(\sigma_{ij} = \text{Cov}(X_i, X_j)\).
Propriétés essentielles

La matrice de covariance est toujours : symétrique (\(\Sigma = \Sigma^T\)), semi-définie positive (toutes les valeurs propres \(\geq 0\)), et diagonalisable dans une base orthonormée (théorème spectral).

Covariance vs Corrélation

La covariance dépend des unités de mesure. La corrélation est une covariance normalisée — elle est toujours entre −1 et 1, ce qui permet de comparer des variables aux échelles différentes. En ACP normée (sur données réduites), on utilise la matrice de corrélation.

\[ r_{ij} = \frac{\sigma_{ij}}{\sigma_i \cdot \sigma_j} = \frac{\text{Cov}(X_i, X_j)}{\sqrt{\text{Var}(X_i)} \cdot \sqrt{\text{Var}(X_j)}} \]
σᵢⱼ : covariance entre \(X_i\) et \(X_j\)  ·  σᵢ : écart-type de \(X_i\)  ·  Diagonale de R = toujours 1  ·  \(r_{ij} \in [-1, 1]\)

4. Centrage & réduction — écriture matricielle

Pourquoi centrer et réduire ?

Centrer : soustraire la moyenne de chaque variable → les données sont centrées en 0. Nécessaire pour que la covariance soit bien calculée. Réduire : diviser par l'écart-type → toutes les variables ont une variance de 1. Indispensable quand les variables ont des unités différentes (ex : taille en cm et poids en kg).

\[ x_{ij}^{\text{centré}} = x_{ij} - \bar{x}_j \qquad x_{ij}^{\text{réduit}} = \frac{x_{ij} - \bar{x}_j}{s_j} \]
x̄ⱼ : moyenne de la variable \(j\)  ·  sⱼ : écart-type de la variable \(j\)

Écriture matricielle du centrage

Formulation matricielle

Soit \(\mathbf{1}_n\) le vecteur colonne de \(n\) uns. La matrice de centrage est \(H = I_n - \frac{1}{n}\mathbf{1}_n\mathbf{1}_n^T\). Alors :

\[ X_c = H \cdot X = \left(I_n - \frac{1}{n}\mathbf{1}_n\mathbf{1}_n^T\right) X \]
H : matrice de centrage (idempotente : \(H^2 = H\), symétrique : \(H = H^T\))  ·  Cette formule permet d'écrire la matrice de covariance comme : \(\Sigma = \frac{1}{n}X^T H X\)
Exemple numérique

Soit \(X = \begin{pmatrix}2\\4\\6\end{pmatrix}\). Moyenne \(\bar{x} = 4\). Données centrées : \(X_c = \begin{pmatrix}-2\\0\\2\end{pmatrix}\). Variance : \(s^2 = \frac{4+0+4}{3} = \frac{8}{3}\). Données réduites : \(X_r = \begin{pmatrix}-2/\sqrt{8/3}\\0\\2/\sqrt{8/3}\end{pmatrix}\).

📌 À retenir / À l'examen

  • Produit scalaire nul \(\Leftrightarrow\) vecteurs orthogonaux \(\Leftrightarrow\) angle de 90°
  • Base orthonormée → \(P^{-1} = P^T\) — simplification fondamentale en ACP
  • Matrice de covariance : \(\Sigma = \frac{1}{n}X_c^T X_c\) — toujours symétrique et semi-définie positive
  • Diagonale de \(\Sigma\) = variances · Hors diagonale = covariances
  • Matrice de corrélation : covariance normalisée, diagonale = 1, valeurs dans \([-1, 1]\)
  • Centrage matriciel : \(X_c = HX\) avec \(H = I - \frac{1}{n}\mathbf{1}\mathbf{1}^T\)
  • ACP sur données non réduites → matrice de covariance · ACP normée → matrice de corrélation
